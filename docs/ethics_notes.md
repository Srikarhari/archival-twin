# Ethics Notes

## Core Principle

This system does NOT infer, predict, or assign race, ethnicity, caste,
nationality, religion, or ANY protected trait to the visitor.

## What the System Does

- Measures geometric resemblance between facial structures
- Returns the closest visual match from a historical archive
- Exposes what machine vision "sees" vs. what it ignores

## What the System Does NOT Do

- Identify, classify, or categorize the visitor
- Make factual claims about the visitor's identity or origin
- Assign demographic labels of any kind

## Archival Context

The archives this system searches were assembled by colonial institutions
whose classification systems reflected the power structures and biases of
their era. The system critiques these systems — it does not reproduce them.
